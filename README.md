# EngineEye

AI and IoT based Predictive Vehicles maintenance system.


---


## Overview

EngineEye is a smart mobile application designed to monitor vehicle health and predict potential issues using AI and IoT concepts. The purpose of this project is to provide real-time insights and early warnings to reduce maintenance costs and improve vehicle safety.

---

## Features

* Predictive maintenance based on AI concepts
* Real-time vehicle monitoring interface
* Sensor-based data simulation (prototype)
* Early fault detection system
* User-friendly mobile application
* Support for both dummy and live data modes

---

## Technologies Used

* Frontend: HTML, CSS, JavaScript
* Mobile App: Capacitor (Android)
* Backend (planned): Node.js
* Database (planned): Supabase / Firebase
* IoT Concept: Sensor-based data simulation

## Purpose

The goal of this project is to develop an affordable and scalable solution for low-cost vehicles that can:

* Predict failures before they occur
* Reduce maintenance expenses
* Improve overall vehicle safety
